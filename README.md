Project Summary:
This project showcases my ability as a Data Analyst to clean, structure, and analyze a real-world data using Microsoft Excel. The objective was to transform raw transactional data into a reliable dataset and extract actionable insights that support business decision-making. I performed data cleaning, transformation, and exploratory data analysis (EDA) using Excel formulas, pivot tables, and charts to analyze sales performance across countries, identify trends over time, and evaluate transaction behavior.

Tools Used:
Microsoft Excel
  * Data cleaning (removal of blanks, and number formating from 1 dp to 2 dp)
  * Data transformation and standardization
  * Date and text formatting
  * Pivot tables and aggregations
  * Charts for trend and performance analysis

Key Insights:
* Cleaning inconsistencies in dates, quantities, and text fields significantly improved analysis accuracy and reporting reliability.

Skills Demonstrated:
* Data Cleaning & Preparation
* Exploratory Data Analysis (EDA)
* Business Insight Generation
* Excel Reporting & Visualization
